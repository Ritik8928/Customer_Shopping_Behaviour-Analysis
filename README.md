#Customer Shopping Behavior Analysis


#Project Overview

This project analyzes customer shopping behavior using transactional data to identify purchasing patterns, customer segments, and product performance. The analysis helps businesses understand customer preferences and improve sales strategies using data-driven insights.

The project combines Python for data preprocessing, SQL for analytical queries, and Power BI for interactive visualization.

Dataset Information

Total Records: 3,900 transactions

Total Features: 18 columns

Missing Values: 37 values in review rating column handled during preprocessing

Dataset Includes

Customer demographics (Age, Gender, Location)

Purchase information (Product Category, Purchase Amount, Season)

Shopping behavior (Discount Applied, Shipping Type, Review Rating)

Subscription status and purchase history

Technologies Used

Python – Data cleaning and feature engineering

Pandas & NumPy – Data processing

SQL (MySQL/PostgreSQL) – Business queries and analysis

Power BI – Data visualization and dashboard

Key Analysis Performed

Revenue analysis by gender

High-spending customers using discounts

Top 5 products based on average rating

Purchase comparison by shipping type

Customer segmentation (New, Returning, Loyal)

Revenue contribution by age group

Dashboard Features

The Power BI dashboard provides interactive insights including:

Total customers and average purchase amount

Revenue distribution by product category

Customer distribution by subscription status

Sales by age group

Interactive filters for gender, category, and shipping type
