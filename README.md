# Customer Shopping Behavior Analysis

## 📊 Project Overview
This project analyzes **customer shopping behavior** using transactional data to identify purchasing patterns, customer segments, and product performance.  
The analysis helps businesses make **data-driven decisions** to improve marketing strategies and increase sales.

---

## 📁 Dataset Information
- **Total Records:** 3,900 purchases  
- **Total Columns:** 18 features  
- **Missing Values:** 37 values in review rating column handled during preprocessing  

### Key Features
- **Age**
- **Gender**
- **Location**
- **Subscription Status**
- **Product Category**
- **Purchase Amount**
- **Discount Applied**
- **Review Rating**
- **Shipping Type**

---

## 🛠️ Technologies Used
- **Python** – Data cleaning and preprocessing  
- **Pandas & NumPy** – Data manipulation  
- **SQL** – Data analysis and business queries  
- **Power BI** – Interactive dashboard and visualization  

---

## 🔎 Key Analysis
- **Revenue by Gender** – Compared revenue generated by male and female customers  
- **Top Rated Products** – Identified products with highest average ratings  
- **Customer Segmentation** – Classified customers into New, Returning, and Loyal  
- **Subscription Analysis** – Compared spending of subscribed vs non-subscribed customers  
- **Revenue by Age Group** – Analyzed which age group contributes most to sales  

---

## 📈 Dashboard Insights
The **Power BI dashboard** provides visual insights such as:

- **Total Customers**

- **Sales by Age Group**
